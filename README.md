# **TCP/IP를 활용한 IoT 스마트 주차장**

* 2025.02.07 ~ 2025.02.27
  
## 1. 기획 


### 1-1. 개발 동기


**가산 디지털 단지 내부의 주차장의 불편함** 


* 주차공간을 찾다가 없어서 나온 적이 많음 
* 어디에, 어떤 **주차 공간**이 **비어있는지** 한 눈에 보길 원함

  
**TCP 통신으로 실시간으로 주차장의 상황 정보를 전달하는 IoT 기반의 스마트 주차장을 만들자!**
